Introduction

FusionPDF is a powerful client-side jQuery plugin for generating PDF documents directly from HTML content or JSON data. It leverages the robust capabilities of pdf-lib and html2canvas to provide a seamless, server-less PDF creation experience.

No Server-Side Processing Needed! Everything happens in the user's browser, making it fast, secure, and easy to integrate into any web application.

Setup & Dependencies

To get started with FusionPDF, you need to include a few libraries in your project. The order is important!

Required Libraries 

<!-- 1. jQuery (3.x+) -->
<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

<!-- 2. PDF-Lib (Core PDF generation) -->
<script src="https://unpkg.com/pdf-lib@1.17.1/dist/pdf-lib.min.js"></script>

<!-- 3. html2canvas (Required for HTML element to PDF) -->
<script src="https://cdnjs.cloudflare.com/ajax/libs/html2canvas/1.4.1/html2canvas.min.js"></script>

<!-- 4. SweetAlert2 (For beautiful alerts - Optional but recommended) -->
<script src="https://cdn.jsdelivr.net/npm/sweetalert2@11"></script>

<!-- 5. FusionPDF Core and jQuery Plugin -->
<script src="./src/core/fusionpdf.js"></script>
<script src="./src/jquery/fusionpdf.jquery.js"></script>

Example 2: Convert JSON Data to PDF

Pass JSON data directly to generate a PDF. The data will be stringified and rendered as pre-formatted text.

Live Demo

The button below will generate a PDF from a predefined JSON object.

jQuery Initialization

$(document).ready(function() {
  const myJsonData = {
    reportTitle: "Quarterly Sales Report",
    period: "Q3 2024",
    author: "Sales Team",
    data: [
      { product: "Widget A", unitsSold: 1500, revenue: 75000 },
      { product: "Widget B", unitsSold: 1200, revenue: 96000 },
      { product: "Accessory X", unitsSold: 3000, revenue: 30000 }
    ],
    summary: "Overall positive growth in Q3."
  };

  $('#generateJsonPdfBtn').fusionPDF({
    json: myJsonData,
    filename: 'json-document.pdf',
    pdfOptions: {
      fontName: 'Courier',
      fontSize: 10,
      margin: { top: 50, right: 30, bottom: 50, left: 30 }
    }
  });
});

Configuration Options

Customize the PDF generation process with these options. Pass them as an object to the .fusionPDF() method.

Option Type Default Description
contentSource String null A CSS selector for the HTML element to convert. If null, the plugin attempts to find a target.
json Object/Array/String null The JSON data to be stringified and added to the PDF.
filename String 'document.pdf' The name of the file to be downloaded.
pdfOptions Object {} An object of options passed directly to the FusionPDFCore engine (e.g., margin, fontSize, fontName).
html2canvasOptions Object {} An object of options passed directly to the html2canvas library (e.g., scale, backgroundColor).

Handling Events

FusionPDF triggers custom events on the jQuery element to notify you of its progress. This is useful for showing loading indicators or handling errors.


// Listen for the success event
$('#generatePdfBtn').on('fusionpdf:success', function(event, data) {
  console.log('PDF generated!', data.blob, data.filename);
  Swal.fire('Success!', 'PDF generated successfully!', 'success');
});

// Listen for the error event
$('#generatePdfBtn').on('fusionpdf:error', function(event, error) {
  console.error('PDF generation failed:', error);
  Swal.fire('Error!', 'PDF generation failed. Check console.', 'error');
});
Pro Tip: You can set up global event handlers on the document to manage all FusionPDF events from one place, as shown in the JSON example's implementation script.
